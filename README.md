# TrackZen

TrackZen is a local-first macOS desktop app for calm, focused time tracking.

This public repository is the download surface for alpha builds. The application source code remains private during the alpha phase.

## Current Alpha

- Version: `0.0.1 alpha`
- Release tag: `0.0.1-alpha`
- Platform: macOS

## Download

- Latest release: `https://github.com/ridala/trackzen-download/releases/latest`
- Direct download for `0.0.1 alpha`: `https://github.com/ridala/trackzen-download/releases/download/0.0.1-alpha/TrackZen-0.0.1-alpha.dmg`

## What TrackZen Does

TrackZen helps freelancers, developers, designers, and consultants:

- track work sessions by client and project
- review daily history and totals
- export CSV reports
- keep data local on the Mac

## Install

1. Download the `.dmg` from the latest release.
2. Open the `.dmg`.
3. Drag `TrackZen.app` into `Applications`.
4. Open TrackZen from `Applications`.

## macOS Note

If macOS warns you because this is an alpha build:

1. Open `System Settings` > `Privacy & Security`.
2. Find the blocked app message for TrackZen.
3. Allow the app to open.
4. Launch the app again.

If the build is signed and notarized before release, this warning may not appear.
